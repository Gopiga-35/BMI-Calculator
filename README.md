#  BMI Calculator App

This is a simple React BMI (Body Mass Index) Calculator built with React and React Router.

You can enter your name, weight (kg), and height (cm) â€” then view your BMI result on a result page with a visual representation.

---

##  Features

- Calculates BMI based on user input
- Shows categorized result (Underweight, Normal, Overweight, Obese)
- Redirects to result page after calculation
- Displays different images for each category
- Uses React Router for navigation
